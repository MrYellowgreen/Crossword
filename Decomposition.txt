Пакет		Описание и назначение

main.py		Консольный интерфейс пользователя.

cell.py		Описание класса Cell - абстрактной клетки кроссворда.

word.py		Описание классов, работающих с "незаполненными словами"
		кроссворда. Содержит классы:
		Word - неизменяемое слово, 
		набор из первой клетки, направления и длины.
		RealWord - совокупность объекта класса Word и 
		списка возможных значений.
		WordsFromGridGetter - класс, который строит список 
		"незаполненных слов" по двумерной сетке символов - 
		стандартному виду кроссворда.

geometry.py	Описание класса Geometry, который по списку объектов 
		класса Word собирает единую структуру.

lexicon.py	Класс Lexicon по списку слов строит структуру, 
		которая по маске слова возвращает список 
		возможных значений.

solver.py	Классы, решающие задачу.
		Solver - класс, непосредственно решающий задачу;
		опирается на Lexicon и Geometry.
		Answer - класс, объекты которого возвращает Solver - 
		описание ответа и методы для его преобразования в текст.

tests.py	Тесты.

stopwatch.py	Секундомер.
